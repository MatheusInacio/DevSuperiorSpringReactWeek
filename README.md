# Passo: Implantação no Heroku
Criar app no Heroku
Provisionar banco Postgres
Definir variável APP_PROFILE=prod
Conectar ao banco via pgAdmin
Criar seed do banco
heroku -v
heroku login
heroku git:remote -a <nome-do-app>
git remote -v
git subtree push --prefix backend heroku main
